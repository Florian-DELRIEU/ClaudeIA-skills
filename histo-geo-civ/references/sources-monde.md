# Sources — International / autres pays

Ce fichier regroupe des sources généralistes valables pour presque tous les pays, plus des sections par pays à étoffer au fil des demandes de Florian (copier le modèle de section ci-dessous pour un nouveau pays).

## Sources généralistes (tous pays)
- Encyclopaedia Britannica (référence encyclopédique anglophone de qualité) : https://www.britannica.com
- Wikipédia (toutes langues — croisement seulement, jamais source unique) : https://www.wikipedia.org
- ONU / United Nations (données démographiques, statistiques mondiales) : https://www.un.org , https://data.un.org
- Banque mondiale (données économiques et démographiques par pays) : https://data.worldbank.org
- CIA World Factbook (fiches pays : géographie, démographie, institutions — à jour et fiable pour les données factuelles de base) : https://www.cia.gov/the-world-factbook
- Eurostat (données statistiques officielles UE) : https://ec.europa.eu/eurostat
- Union européenne (institutions, droit européen) : https://european-union.europa.eu

## Notes d'usage
- Pour un pays non encore listé ici, chercher directement le site du parlement/gouvernement national et l'institut statistique national ; les ajouter ensuite à ce fichier.
- Le CIA World Factbook est une bonne source de dépannage rapide pour la géographie/démographie de n'importe quel pays quand aucune source officielle locale n'a encore été référencée.
