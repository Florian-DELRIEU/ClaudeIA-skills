# Lore Personnel — Personnages [ZV]

Contient les personnages originaux créés par l'utilisateur, ainsi que les versions modifiées de personnages canon existants.

## Format d'entrée

```
### [Nom du personnage]
- **Statut** : Original | Version modifiée de [Nom canon]
- **Affiliation** : [Faction / organisation]
- **Relation au canon** : Étend | Remplace | Comble un vide | Contredit
- **Contenu** : [Biographie, rôle, caractéristiques]
- **Divergence canon** : [Si version modifiée d'un perso canon, ce que dit la version officielle]
```
