## Workflow d'ajout

Quand l'utilisateur demande d'ajouter du lore :
1. Lire cet index pour vérifier si une entrée similaire existe déjà
2. Ouvrir le fichier de catégorie approprié
3. Ajouter l'entrée au format standard (voir en-tête de chaque fichier)
4. Ajouter une ligne dans la table ci-dessus
5. Repackager sur commande `/maj`
