# Sources — France

## Histoire
- Assemblée nationale — Histoire et patrimoine : https://www.assemblee-nationale.fr/dyn/histoire-et-patrimoine
- Herodote.net (chronologies détaillées, bonne base mais site privé — vérifier avec une source officielle sur les points sensibles) : https://www.herodote.net
- Gallica / BnF (archives et documents d'époque numérisés) : https://gallica.bnf.fr
- Archives nationales : https://www.archives-nationales.culture.gouv.fr
- INA (archives audiovisuelles, utile pour histoire contemporaine 20e-21e s.) : https://www.ina.fr
- Wikipédia FR — bon point de départ et croisement, jamais source unique : https://fr.wikipedia.org

## Géographie
- INSEE (données démographiques, économiques, territoriales officielles) : https://www.insee.fr
- IGN — Institut national de l'information géographique et forestière (cartographie, données territoriales officielles) : https://www.ign.fr
- data.gouv.fr (données publiques ouvertes) : https://www.data.gouv.fr

## Sciences civiques / institutions
- vie-publique.fr (Direction de l'information légale et administrative — explications institutionnelles officielles) : https://www.vie-publique.fr
- Légifrance (textes de loi, Constitution, codes) : https://www.legifrance.gouv.fr
- Conseil constitutionnel : https://www.conseil-constitutionnel.fr
- Assemblée nationale : https://www.assemblee-nationale.fr
- Sénat : https://www.senat.fr
- Conseil d'État : https://www.conseil-etat.fr
- Élysée (présidence de la République) : https://www.elysee.fr

## Notes d'usage
- Pour toute question institutionnelle (élections, pouvoirs, Constitution), Légifrance ou vie-publique.fr priment sur Wikipédia.
- Pour les dates et faits historiques, croiser Assemblée nationale / Herodote.net / Gallica ; Wikipédia sert à dégrossir seulement.
- Pour les chiffres (population, territoires), toujours INSEE ou IGN en source de référence.
