# Panel de sources de repère

Ce panel est un **point de départ non exhaustif**. Claude complète toujours par recherche web selon le sujet, et n'est pas tenu de se limiter à cette liste. L'objectif est de garantir, pour chaque sujet de Une, la présence d'au moins un angle de gauche, un du centre, et un de droite.

Le classement politique est indicatif (positionnement éditorial généralement admis) et sert uniquement à équilibrer les angles — il ne constitue pas un jugement de Claude.

## Ground News

Agrégateur qui classe automatiquement la couverture d'un sujet en **Left / Center / Right** et signale les "blindspots" (sujets sur-couverts d'un seul bord).

Usage :
- Rechercher `ground news [sujet]` pour trouver la page d'agrégation du sujet.
- En tirer : la répartition de couverture, l'éventuel biais dominant, les titres croisés.
- Reporter ce constat de façon neutre dans la Note de rédaction.

## Règle d'équilibre

Pour un sujet **franco-français** : privilégier le panel français, avec au moins un titre de chaque bord.
Pour un sujet **international** : croiser presse française + presse internationale des deux bords + Ground News.
Si un bord politique n'a pas couvert le sujet, le noter — c'est en soi une information.
