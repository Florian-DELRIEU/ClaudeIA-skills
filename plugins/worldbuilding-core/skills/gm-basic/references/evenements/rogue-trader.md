# Table d'événements — Rogue Trader

Seeds spécifiques à l'univers Rogue Trader (FFG / WH40K).
S'utilise en remplacement ou en complément de `generique.md`.
L'ancrage narratif reste obligatoire — voir `generique.md` pour la règle complète.

---

## Ancrage dans l'univers RT

Avant d'instancier un seed, ancrer sur l'un des éléments établis :

- **Personnages** : officiers nommés, PNJ récurrents, équipage notable, contacts
- **Factions** : maisons marchandes, guildes, Adeptus présents, factions xénos actives
- **Lieux** : mondes visités, épaves cartographiées, systèmes parcourus
- **Fils narratifs** : conflits en cours, dettes ouvertes, rivalités établies, secrets connus

Un seed sans ancrage possible devient une **rumeur de secteur** — information circulante,
sans confrontation directe, qui peut devenir un ancrage futur.

## Utilisation combinée

Pour chaque tirage d'événement, le MJ :

1. Détermine la **catégorie** (Menace / Complication / Opportunité / Révélation / Neutre)
2. Choisit le **domaine** le plus cohérent avec la situation actuelle de la Flotte
3. Sélectionne le **seed** le mieux ancrable dans l'histoire en cours
4. **Instancie** avec des noms, lieux et factions déjà établis

Plusieurs seeds peuvent se combiner si le tirage produit 2–3 événements :
un seed V + un seed W sur la même fenêtre crée naturellement une tension bicéphale
(problème matériel *et* problème occulte) qui se renforce mutuellement.
