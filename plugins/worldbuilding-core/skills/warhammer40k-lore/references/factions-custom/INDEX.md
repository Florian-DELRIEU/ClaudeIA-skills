## Notes d'utilisation

- Toutes les entrées sont **[ZV]** (lore personnel, non-canon officiel).
- En cas de conflit avec le canon, la version [ZV] est prioritaire, avec la version canon entre parenthèses.
- Chaque fichier est un document vivant — statut par section indiqué dans le fichier.

## Workflow d'ajout d'une nouvelle faction

1. Créer `nom-de-faction.md` dans ce dossier
2. Ajouter une ligne dans la table ci-dessus
3. Mettre à jour `personal-lore/INDEX.md` avec une entrée courte pointant ici
4. Mettre à jour `SKILL.md` si nécessaire
5. Repackager sur commande `/maj`
